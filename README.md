# PlayerForStream

트위치 방송을 볼 수 있는 간단한 안드로이드 어플.

트위치 공식 앱은 방송 미리보기가 너무 작아 불편해서 만들었습니다.

주요 특징 : 
  채팅창에서 관리자 배지가 달린 채팅을 편의를 위해 오른쪽으로 배치.
